ES5 -- Objects and Functions


  • Every Js object has a prototype property, which makes inheritance possible in Js

  • The prototype property of an object is where we put methods and properies that we want other objects to inherit

  • The constructor's prototype property is NOT the prototype of the constructor itself, it’s the prototype of ALL instances that are created through it

  • When a certain method or property is called, the search starts in the object itself, and if it cannot be found, the search moves on to the object’s prototype. This continues until the method is found: prototype chain
